# 暗流涌动（TB）技能覆盖清单

## 镇民
- Washerwoman：首夜获知“2人中有1人是某镇民”。
- Librarian：首夜获知“2人中有1人是某外来者”，或0外来者提示。
- Investigator：首夜获知“2人中有1人是某爪牙”。
- Chef：首夜获知邪恶相邻对数。
- Empath：每夜获知两侧存活邻居中邪恶数量。
- Fortune Teller：每夜查验2人，含红鲱鱼机制。
- Undertaker：每夜（首夜除外）获知当天被处决者身份。
- Monk：每夜（首夜除外）守护1名非自己玩家免受恶魔击杀。
- Ravenkeeper：夜死后查验1名玩家身份。
- Virgin：首次被提名时，若提名者为镇民则提名者立即被处决。
- Slayer：白天一次公开开枪，命中恶魔目标则其死亡。
- Soldier：免疫恶魔击杀（被中毒/醉酒时失效）。
- Mayor：
  - 仅剩3人且当天无处决时善良胜。
  - 夜间被恶魔击杀时可能转移死亡。

## 外来者
- Butler：每夜选主人，次日仅能在主人投票时投票。
- Drunk：以“伪装镇民”身份参与，能力按醉酒失真处理。
- Recluse：可注册为邪恶/爪牙/恶魔。
- Saint：被处决时触发善良失败（特殊胜利优先级已接入）。

## 爪牙
- Poisoner：每夜中毒1人（影响当夜 + 次日日间）。
- Spy：每夜查看魔典摘要；可注册为善良/镇民/外来者。
- Scarlet Woman：恶魔死亡且场上≥5存活时接任恶魔。
- Baron：开局外来者 +2、镇民 -2。

## 恶魔
- Imp：每夜（首夜除外）击杀1人；可自杀并传位给爪牙。

## 胜负规则
- 特殊胜负优先结算。
- 同时满足善恶胜利时，善良优先。
- 通用胜负：恶魔死亡则善良胜；场上仅剩两人且善良未先获胜则邪恶胜。
